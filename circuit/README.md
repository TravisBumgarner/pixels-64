# Circuit

## Circuit Overview

Each board is a 4x4 

## Versions

Each version has two folders inside. 
- `kicad` contains the KiCad project files.
- `jlcpcb` contains the files sent to JLCPCB for order.

### V1.0

Initial PCB design, ordered via JLCPCB. 

Specs
- Dimension: 100mm x 100mm
- Layers 2
- PCB Thickness: 1.6mm
- PCB Color: Black
